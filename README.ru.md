# Interactive Booking Scheduler

🇷🇺 Русский | 🇬🇧 [English](README.md)

Высокопроизводительный планировщик бронирований, построенный на React Native, Skia, Reanimated и Gesture Handler.

Проект демонстрирует архитектурные и инженерные подходы, используемые при разработке календарей, систем резервирования ресурсов и приложений для управления расписанием.

---

## Возможности

### Рендеринг

- GPU-ускоренный рендеринг через React Native Skia
- Виртуальная система координат
- Закрепленные заголовки и боковая панель ресурсов
- Индикатор текущего времени
- Масштабируемая временная шкала

### Взаимодействие

- Панорамирование (Pan)
- Масштабирование (Pinch-to-Zoom)
- Перетаскивание бронирований
- Изменение длительности бронирования
- Создание бронирования по нажатию
- Разрешение конфликтов жестов

### Бизнес-логика

- Проверка пересечений бронирований
- Ограничение минимальной длительности
- Минимальные интервалы между бронированиями
- Ограничения рабочего дня
- Привязка бронирований к ресурсам

---

## Производительность

В проекте используются:

- React Native Skia для GPU-рендеринга
- Reanimated Worklets для вычислений на UI Thread
- Shared Values для обновления viewport без участия React Render Cycle
